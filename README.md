guest_book 
============================

ТРЕБОВАНИЯ
------------

PHP 5.6.0

MySQL 4.5.2

УСТАНОВКА
------------
### Файловая система
Поместите содержимое репозитория в корень веб-сервера.

Для входа в админ-панель необходимо пройти http - авторизацию:
admin:admin

### База данных

Разверните дамп базы (/quest_book.sql) данных из корневой директории сайта на сервере mysql.
Если Вы используете нестандартную конфигурацию внесите соответствующие изменения в файл /database.php:

define('MYSQL_SERVER', 'Сервер БД');

define('MYSQL_USER', 'Пользователь БД');

define('MYSQL_PASSWORD','Пароль');

define('MYSQL_DB', 'Имя базы данных(по умолчанию: guest_book)');
